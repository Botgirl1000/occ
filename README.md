## 📂 What is occ?

occ is a simple tool to help you understand and explore documents and code in your files. It works with many common document types like Word, PDF, Excel, and PowerPoint. You can use it to count words, pages, and analyze the structure of documents in your folders. Occ also helps extract information from code repositories, making it easier to browse and manage your projects.

You do not need any programming skills to use occ. It runs on Windows and works through an easy command-line interface.

## ⚙️ How occ Works

occ runs from the command line. This means you type instructions in a window called "Command Prompt." Don't worry — instructions are simple and clear. You can use pre-set commands to get data from your documents or code without learning programming.

When you start occ, you tell it where your files are. Then it reads them and provides counts and lists that tell you about the content. For example, you can find out how many pages are in a folder of PDFs or how many words are inside Word documents.

## 🖥️ Running occ on Windows

### Step 1: Open Command Prompt

- Press the Windows key on your keyboard.
- Type `cmd` and hit Enter.
- A black window will open. This is the Command Prompt.

### Step 2: Check occ Installation

Type this command and press Enter:

```
occ --help
```

You should see a list of commands and options. This means occ is installed correctly.

### Step 3: Use occ to Analyze Documents

To check all documents in a folder, use this example command:

```
occ analyze C:\Users\YourName\Documents
```

Replace `C:\Users\YourName\Documents` with the folder path where your documents are stored.

Occ will scan the files and print a report showing word counts, page numbers, and other useful details.

## 📖 Common Commands

- **Analyze a folder**  
  ```
  occ analyze <folder-path>
  ```  
  Scan all supported files in the folder and show metrics.

- **Count pages in PDFs**  
  ```
  occ count-pdf-pages <file-path>
  ```  
  Show the number of pages in a specific PDF.

- **Count words in Word documents**  
  ```
  occ count-words <file-path>
  ```  
  Show how many words are in a single Word file.

- **Extract document structure**  
  ```
  occ extract-structure <file-path>
  ```  
  Show the outline or headings found inside a document.

- **Explore code repository**  
  ```
  occ explore-code <folder-path>
  ```  
  Produce insights about code files in a project folder.
